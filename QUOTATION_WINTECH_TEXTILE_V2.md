## Project Summary

Extension of WinTech ERP application to support textile production line operations, including:

- Operation-based workflow system for all product families
- Talon (barcode label) system for operator tracking and anomaly management
- Kwantum (KW) client XML file integration (NEV/IC2S, PI2S, SA2S, GOMV/VOWV, BDC Status)
- RFID printing capability for packaging (Zebra ZT411)
- Accessory stock management
- KW textile invoicing
